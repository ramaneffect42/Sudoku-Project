What is Sudoku?

Sudoku (japanese: 数独, romanized: sūdoku, lit. 'digit-single'; originally called Number Place) is a logic-based, combinatorial number-placement puzzle. In classic Sudoku, the objective is to fill a 9 × 9 grid with digits so that each column, each row, and each of the nine 3 × 3 subgrids that compose the grid (also called "boxes", "blocks", or "regions") contains all of the digits from 1 to 9. The puzzle setter provides a partially completed grid, which for a well-posed puzzle has a single solution.

Rules for Sudoku:

Here are the basic rules of sudoku

  1. Complete the grid: The goal is to fill a 9x9 grid with digits.
  2. Use digits 1-9: You may only use numbers from 1 to 9 in the grid.
  3. No repeats in rows: Each horizontal row must contain each digit from 1 to 9 exactly once.
  4. No repeats in columns: Each vertical column must contain each digit from 1 to 9 exactly once.
  5. No repeats in 3x3 boxes: The 9x9 grid is divided into nine 3x3 boxes. Each of these boxes must also contain each digit from 1 to 9 exactly once.

Rough Summary

The challenge of Sudoku is to populate an 81-cell grid that is sectioned into nine 3x3 blocks. The game’s core constraint applies equally to every horizontal row, vertical column, and 3x3 block: each must contain a complete set of the digits 1 through 9, with no number repeated. Using logic and reasoning, your goal is to deduce the single correct placement for a digit in every empty cell. The puzzle is won once the entire grid is correctly filled.

Rough Summary

To solve a Sudoku, ensure each row, column, and 3x3 box contains the numbers 1 through 9 without any repeats. Use the "cross-hatching" technique, a process of elimination, to deduce where numbers must go based on existing placements. Work systematically through each box, repeating the process as filled squares reveal new solutions. For difficult squares, try counting through the possible numbers (1-9) to see which one is missing from the corresponding row, column, and box.
